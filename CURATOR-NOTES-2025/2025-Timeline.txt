Maintainer history
- Chemtool was created by Thomas Volk while he was a chemistry and biology student at Ulm University; he maintained it through version 1.1.1, establishing the base GUI and feature set. The main.c header pins that first code drop to January 1998.【chemtool-1.6.14/README】【chemtool-1.6.14/main.c】
- Dr. Martin Kroeker has been the primary developer since version 1.1.2, stewarding all subsequent releases, handling packaging, and serving as the public contact at the University of Freiburg. The same header records his GTK rewrite and feature work across 1999–2007 and follow-up maintenance pushes in 2010, 2011, and 2013.【chemtool-1.6.14/README】【chemtool-1.6.14/chemtool.spec】【chemtool-1.6.14/main.c】

Release & contributor chronology
- 1.1.x branch – Transition to GTK UI and early feature growth. Robert McMullen’s ListTree widget powered the file dialog, while Thomas Volk’s final releases introduced core bond manipulation tools before Martin Kroeker assumed maintainership.【chemtool-1.6.14/ChangeLog】
- 1.2 alpha series – Martin Kroeker focused on stability, export capabilities, and undo support using Matt Kimball’s GPL’d libundo library. Radek Liboska began contributing heavily with xfig/export fixes, translation groundwork, templates, and example molecules.【chemtool-1.6.14/ChangeLog】
- 1.3 line – Usability refinements (iconic toolbars, template overhaul) credited to Radek Liboska, with localization and template support drawing on contributions from Michael Banck. cht integration (molecular weight helper) was added, authored by Radek Liboska.【chemtool-1.6.14/ChangeLog】
- 1.4 development (2001) – New GTK menu bar (Michael Banck), Bézier curves, bracket tools, and Babel-based import groundwork; translations expanded (Jerome Pansanel – French). Undo portability improvements pulled in outside patches.【chemtool-1.6.14/ChangeLog】
- 1.5 series (2004–2005) – Major import/export push driven by Michael Banck (MDL molfile, PDB parsing, print support). Radek Liboska and Flavio Matsumoto extended documentation and translations; Matt Kimball’s undo stack was fully embedded.【chemtool-1.6.14/ChangeLog】
- 1.6 alpha–stable (2004 onward) – Long arc of modernization under Martin Kroeker:
  - Early alphas added Babel import, EMF export, grid backdrops, color, and spline enhancements with help from Pawel Golaszewski (DESTDIR packaging), Nakata Maho (mmap portability), Christian Weisgerber (cht cleanups), Daniel Uppström (UI polish), and Fabio Rainone (legacy UNIX mmap fixes).【chemtool-1.6.14/ChangeLog】
  - Later alphas stabilized GTK2 builds, improved SVG output (Radek Liboska), enriched translations (Jerome Pansanel, Nikodem Kuznik, Michael Shigorin), tweaked bond rendering (Michael Banck, Soeren Kristensen), and added background grids and attachment markers.【chemtool-1.6.14/ChangeLog】
  - Stable 1.6 releases continued through 1.6.14 with sustained maintenance: new export formats (PNG, Asymptote, Babel-backed), Unicode symbol support, batch mode, translations (Nuno Azoia – European Portuguese; Svetoslav Stefanov – Bulgarian; Micha Smoczyk – Polish), build fixes (Dominik Mierzejewski, Daniel Leidert), and UI refinements (Sebastian Koppehel).【chemtool-1.6.14/ChangeLog】【chemtool-1.6.14/NEWS】

Packaging timeline with dates
The RPM spec’s changelog anchors release activity with explicit dates, confirming Martin Kroeker as the packager from 2004 onward and documenting earlier packaging by Radek Liboska:
- 2001‑07‑27 – Radek Liboska packaged Chemtool 1.4 for Red Hat; KDE/GNOME integration work followed in 2002.【chemtool-1.6.14/chemtool.spec】
- 2003‑07 – Radek adopted Red Hat 9 desktop conventions and generalized install paths.【chemtool-1.6.14/chemtool.spec】
- 2004‑05 to 2005‑07 – Martin Kroeker took over packaging, updating build scripts, translations, and documentation with each 1.6.x point release.【chemtool-1.6.14/chemtool.spec】
- 2006‑2013 – Martin maintained regular spec updates through versions 1.6.5–1.6.14, including build-system refreshes (autoconf 2.69 on 2013‑08‑12 for ARM64 support).【chemtool-1.6.14/chemtool.spec】
